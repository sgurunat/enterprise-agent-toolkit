# Intel® AI Enterprise Agent Toolkit - Scripts

This directory contains utility scripts for the Intel® AI Enterprise Agent Toolkit.

## Scripts Overview

### `keycloak-fetch-client-secret.sh`
Retrieves Keycloak client secrets for authentication setup.

### `keycloak-realmcreation.sh`
Automates Keycloak realm creation and configuration.

### `generate-token.sh`
Generates authentication tokens for API access.

### `generate-vault-secrets.sh`
Generates vault secrets for secure credential storage.
